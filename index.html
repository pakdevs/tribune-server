News Proxy is deployed

This project exposes API routes for the Pakistan Tribune app.

API endpoints:

  • https://tribune-server.vercel.app/api/world (NewsAPI top headlines, world)
  • https://tribune-server.vercel.app/api/pk (NewsAPI everything: Pakistan-focused; domain-scoped by default)
  • https://tribune-server.vercel.app/api/search?q=test (supports domains, from, to)

Examples:

  • https://tribune-server.vercel.app/api/world?page=1&pageSize=20
  • https://tribune-server.vercel.app/api/pk?page=1&pageSize=20
  • https://tribune-server.vercel.app/api/pk?domains=dawn.com,geo.tv&mode=extend&page=1&pageSize=20 (override/extend default PK domains)
  • https://tribune-server.vercel.app/api/search?q=pakistan&page=1&pageSize=20
  • https://tribune-server.vercel.app/api/search?q=economy&domains=theguardian.com,reuters.com&from=2024-01-01&to=2024-12-31&page=1&pageSize=20

Source endpoints

General format:

  • /api/world/source/{slug}?name=...
  • /api/pk/source/{slug}?name=...

Notes:
• Category aliases: politics/world → general, tech → technology, sci → science, biz → business.
• Pagination: page (>=1), pageSize (1–100).
• Provider: NewsAPI (Top headlines and Everything).
• PK endpoints accept domains= (comma-separated) and mode=extend|replace to adjust the curated domain set.
• Search supports domains, from, and to.
• Source endpoints accept name and optional domain for better matching.
• For testing add ?debug=1&nocache=1 to view request and bypass cache.

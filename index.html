News Proxy is deployed

This project exposes API routes for the Pakistan Tribune app.

API endpoints:

  • /api/top (US default, all categories)
  • /api/world (multi-provider, world news)
  • /api/pk (multi-provider, Pakistan)
  • /api/search?q=test

Examples:

  • /api/top?page=1&pageSize=20
  • /api/top?category=technology&page=2
  • /api/world?page=1&pageSize=20
  • /api/pk?page=1&pageSize=20
  • /api/search?q=pakistan&page=1&pageSize=20

Categories (World)

  • /api/pk/category/health
  • /api/pk/category/science
  • /api/pk/category/sports
  • /api/pk/category/technology

Notes:
• Category aliases: politics/world → general, tech → technology, sci → science, biz → business.
• Pagination: page (>=1), pageSize (1–100).
• Provider fallback & rotation reduce rate-limit impact.
• PK endpoint final fallback: NewsAPI Everything (q=Pakistan).
• Add ?debug=1 to any endpoint (dev only) to inspect upstream provider.

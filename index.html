News Proxy is deployed

This project exposes API routes for the Pakistan Tribune app.

API endpoints:

  • /api/world (NewsAPI top headlines, world)
  • /api/pk (NewsAPI everything: Pakistan-focused; domain-scoped by default)
  • /api/search?q=test (supports domains, from, to)

Examples:

  • /api/world?page=1&pageSize=20
  • /api/pk?page=1&pageSize=20
  • /api/pk?domains=dawn.com,geo.tv&mode=extend&page=1&pageSize=20 (override/extend default PK domains)
  • /api/search?q=pakistan&page=1&pageSize=20
  • /api/search?q=economy&domains=theguardian.com,reuters.com&from=2024-01-01&to=2024-12-31&page=1&pageSize=20

Source endpoints

General format:

  • /api/world/source/{slug}?name=...
  • /api/pk/source/{slug}?name=...

Categories (Pakistan)

  • /api/pk/category/entertainment
  • /api/pk/category/general
  • /api/pk/category/health
  • /api/pk/category/science
  • /api/pk/category/sports
  • /api/pk/category/technology

Notes:
• Category aliases: politics/world → general, tech → technology, sci → science, biz → business.
• Pagination: page (>=1), pageSize (1–100).
• Provider: NewsAPI (Top headlines and Everything).
• PK endpoints accept domains= (comma-separated) and mode=extend|replace to adjust the curated domain set.
• Search supports domains, from, and to.
• Source endpoints accept name and optional domain for better matching.
• For testing add ?debug=1&nocache=1 to view request and bypass cache.
